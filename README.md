## Build with: ```fakeroot dpkg-deb --build (folder name)``` 

## You can download already built package here: [sourceforge](https://sourceforge.net/projects/hse-settlers-of-catan/files/Catan_1.0_all.deb/download)

- ### Run ```sudo apt install ./Catan_1.0_all.deb``` to install game

- ### Run ```catan``` to run game

- ### Run ```sudo apt remove catan``` to remove game
